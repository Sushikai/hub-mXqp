## 一、优化方案设计

### 1. 输入层优化（Start节点）

新增变量：
- age（年龄，number类型）

完整输入结构：
- sex（性别）
- age（年龄）
- height（身高）
- weight（体重）

---

## 二、优化后 Dify YAML

```yaml
app:
  name: 应用-体脂健康建议-优化版
  mode: workflow

workflow:
  graph:
    nodes:

    - id: start
      type: start
      data:
        variables:
        - label: 性别
          type: select
          variable: sex
          options: [男, 女]
          required: true

        - label: 年龄
          type: number
          variable: age
          required: true

        - label: 身高(cm)
          type: number
          variable: height
          required: true

        - label: 体重(kg)
          type: number
          variable: weight
          required: true

    - id: calc
      type: code
      data:
        code: |
          def main(sex, age, height, weight):
              # 计算 BMI
              bmi = weight / (height / 100) ** 2

              # Deurenberg 体脂率公式
              if sex == '男':
                  bf = 1.2 * bmi + 0.23 * age - 16.2
              elif sex == '女':
                  bf = 1.2 * bmi + 0.23 * age - 5.4
              else:
                  bf = 0

              # 合理范围限制
              bf = max(0, min(100, bf))

              return {
                  "bmi": round(bmi, 2),
                  "body_fat": round(bf, 2)
              }

    - id: llm
      type: llm
      data:
        prompt: |
          你是一名专业营养师，请基于以下数据分析用户健康状态：

          【用户信息】
          性别：{{sex}}
          年龄：{{age}}

          【身体指标】
          BMI：{{bmi}}
          体脂率：{{body_fat}}

          请输出：
          1. 身体状态（偏瘦 / 正常 / 超重 / 肥胖）
          2. 健康风险分析
          3. 饮食建议（具体）
          4. 运动建议（具体）

    - id: template
      type: template-transform
      data:
        template: |
          ## 用户信息
          - 性别：{{ sex }}
          - 年龄：{{ age }}

          ## 身体指标
          - BMI：{{ bmi }}
          - 体脂率：{{ body_fat }}

          ## 健康建议
          {{ text }}

    - id: end
      type: end
```
