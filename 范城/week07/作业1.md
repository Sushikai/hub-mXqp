# BERT文本分类与实体识别总结

## 1. 关系

BERT文本分类（意图识别）和实体识别（槽位填充）采用**共享BERT编码器的联合训练架构**：

- **共享特征提取器**：两个任务共享同一个BERT模型，将输入文本编码为向量表示
- **并行双任务输出**：
  - 意图识别：使用BERT的`pooler_output`（句子级768维向量）进行分类
  - 实体识别：使用BERT的`token_output`（token级序列输出）进行序列标注

## 2. 使用的Loss

两个任务**都使用CrossEntropyLoss**：

- **意图识别loss**：针对句子级分类的交叉熵损失
- **实体识别loss**：针对每个token的实体标签预测的交叉熵损失（只计算有效token，忽略padding部分）
- **联合优化**：总loss = seq_loss + token_loss，同时优化两个任务

## 3. loss = seq_loss + token_loss 的坏处以及解决方案

### 数值尺度不一致
- 两个任务的loss可能处于不同的数值范围
- 例如：seq_loss可能是2-3，token_loss可能是20-30
- 简单相加会导致大loss的任务主导整个训练过程

### 解决方案：加权Loss
```python
loss = α * seq_loss + β * token_loss
```
- 根据任务重要性手动设置权重
- 通过实验调整α和β的值
- 简单但需要大量调参
