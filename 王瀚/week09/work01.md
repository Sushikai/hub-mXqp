在 Dify 中改进体脂建议应用并增加“年龄”输入，可以显著提升建议的个性化和准确性。因为不同年龄段（如青少年、成年人、老年人）对体脂率的健康标准和代谢需求完全不同。

以下是具体的改进步骤，分为 **变量配置**、**提示词优化** 和 **前端设置** 三个部分：

### 第一步：添加“年龄”变量

首先，需要让应用能够接收用户的年龄数据。

1. 进入 Dify 应用的 **“设置”** 或 **“编排”** 页面。
2. 在 **“功能”** 或 **“变量”** 区域（通常在提示词编辑器的上方或左侧），找到变量列表。
3. 点击 **“添加变量”**。
4. 配置如下：
   - **变量名:** `age`
   - **显示名称:** 用户年龄
   - **类型:** 数字 或 段落 (建议选“数字”以限制输入)
   - **必填:** 是
   - **默认值:** (可选，例如留空或填 25)

### 第二步：优化提示词

修改“提示词”区域，告诉 AI 如何利用这个新的 `age` 变量来调整建议。

**修改前的逻辑可能只是：** “根据体脂率给出建议。”
**修改后的逻辑应该是：** “结合年龄、性别和体脂率，参考该年龄段的健康标准给出建议。”

请参考以下提示词模板进行更新：

#### 系统提示词模板

```markdown
# 角色
你是一位专业的健康管理专家和健身教练。你擅长根据用户的身体数据提供科学、客观的体脂率分析和改善建议。

# 输入数据
- 用户性别：{{gender}}
- 用户年龄：{{age}} 岁
- 用户当前体脂率：{{body_fat_percentage}}%

# 任务
请根据上述输入数据，为用户生成一份详细的体脂健康报告。

# 分析逻辑
1. **评估现状**：
   - 结合用户的 **年龄 ({{age}})** 和 **性别**，判断其体脂率处于什么水平（例如：过低、健康/理想、偏高、肥胖）。
   - *注意：请考虑到随着年龄增长，人体基础代谢率会下降，且老年人维持稍高一点的体脂率可能更利于健康，而年轻人则更追求肌肉线条。*

2. **健康风险**：
   - 如果体脂率异常，请指出可能面临的健康风险（如心血管疾病、代谢综合征等）。

3. **改进建议**：
   - **饮食建议**：针对该年龄段代谢特点，给出具体的饮食调整方案（例如：增加蛋白质摄入、控制碳水等）。
   - **运动建议**：推荐适合的运动类型（有氧、无氧、HIIT等）及频率。
   - **生活习惯**：睡眠、压力管理等方面的建议。

# 输出要求
- 语气要专业、鼓励性强且充满关怀。
- 使用 Markdown 格式，利用 **加粗**、列表和标题使内容易读。
- 如果输入数据缺失，请礼貌地要求用户补充。
```

### 第三步：优化前端体验

在“预览”或“发布”后的界面中，确保用户能清楚地看到输入框。

1. **输入表单排序**：建议将“年龄”放在“性别”之后，“体脂率”之前。逻辑顺序通常是：性别 -> 年龄 -> 身体数据。
2. **添加占位符提示**：在年龄输入框的提示文本中，可以写上“请输入您的周岁年龄，例如：30”。
